# دليل تشغيل تطبيق مراقبة تليجرام (Telegram Monitor PWA)

## المتطلبات
- Node.js (الإصدار 16 أو أحدث)
- MongoDB (مثبت محلياً أو حساب Atlas)
- توكن بوت تليجرام (Telegram Bot Token)

## 1. إعداد الواجهة الخلفية (Backend)
1. افتح المجلد `backend`:
   ```bash
   cd backend
   ```
2. قم بتثبيت المكتبات المطلوبة:
   ```bash
   npm install
   ```
3. تأكد من إعداد ملف `.env`:
   - تأكد من صحة رابط قاعدة البيانات `MONGO_URI`.
   - ضع توكن البوت `BOT_TOKEN` ومعرف المجموعة `GROUP_ID`.
4. شغل السيرفر:
   ```bash
   npm start
   ```
   سيعمل السيرفر على الرابط: `http://localhost:5000`.

## 2. إعداد الواجهة الأمامية (Frontend)
1. افتح نافذة تيرمنال جديدة واذهب لمجلد `frontend`:
   ```bash
   cd frontend
   ```
2. قم بتثبيت المكتبات:
   ```bash
   npm install
   ```
3. شغل التطبيق:
   ```bash
   npm run dev
   ```
   سيعمل التطبيق على الرابط: `http://localhost:5173`.

## 3. ربط البوت محلياً (للتجربة)
لكي يستقبل البوت الرسائل وأنت تعمل على جهازك، يجب استخدام خدمة مثل `ngrok`.
1. حمل وثبت `ngrok`.
2. شغل الأمر:
   ```bash
   ngrok http 5000
   ```
3. انسخ رابط HTTPS الظاهر (مثلاً `https://xxxx.ngrok-free.app`).
4. ضع هذا الرابط في إعدادات التطبيق في خانة `Webhook URL` (مع إضافة `/webhook` في نهايته).
